 Análise de Perfil de Clientes e Veículos

Este projeto apresenta uma análise exploratória de dados focada no perfil de clientes e nos veículos mais visitados em um funil de vendas do setor automotivo. O objetivo é transformar dados brutos em insights visuais, apoiando a tomada de decisão estratégica.

O dashboard consolida informações demográficas, socioeconômicas e comportamentais dos usuários, permitindo uma visão clara sobre quem são os clientes e quais veículos despertam maior interesse.
